Hi! Thanks a lot for helping us in classifying the incidents collected on Ushahidi platform.

The application is really simple: it has only 3 steps. The first one loads a report and asks you to categorize it according to different categories and its sub-categories.

In the second step, you have to find the location from the report. You will be able to place it in a map

Usually you will:

  • find the name of the city
  • paste it in the input field and
  • click in the search button Search

The map will search that location and add a blue marker, then you have to place a marker in the position that the tweet is referencing.

To place the marker:

  • click in the button: Add a marker
  • hover the mouse to place it at the location (a green typhoon marker will appear)
  • click to set the coordinates
Tip: If you need to fix the position you can drag and drop it again.

Once you have chosen one of the categories and located the position of the report, you will be able to review the classification in the third and final step. Click in the Submit classification button and you will have classified one report.

That's all!
